# Reactive Agent

A minimal **reactive agent** implemented in Python that interacts with a simple traffic light environment.  
This project demonstrates the core principle of reactive agents: perceiving the environment and acting immediately based on the current state.

---

## Features
- Clean, modular Python code
- Enum-based state representation for clarity
- Easily extendable to more complex environments
- No external libraries required
